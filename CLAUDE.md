# 專案:台股交易系統(Streamlit)

## 專案概述
- 台股分析平台,部署於 Streamlit Community Cloud
- 頁面:選股系統(動能選股/破底翻掃描)、總經儀表板、新聞監控、
  散戶指標、個股監控、板塊熱力圖、產業研究、Podcast 整理、AI 問答
- 使用者介面語言:繁體中文(台灣用語)

## 技術約定
- Python 3.x + Streamlit
- 所有外部 API 呼叫必須包 try/except,失敗時顯示友善錯誤訊息,不可讓頁面整個掛掉
- 所有行情/資料抓取函式必須加 @st.cache_data(ttl=...),TTL 依資料性質決定:
  - 盤中報價:60~300 秒
  - 月營收、財報、法人籌碼:86400 秒(一天)
- API 金鑰一律放 st.secrets,絕不可寫死在程式碼或 commit 進 git
- 修改前先讀完相關檔案;改動保持最小範圍,不要順手重構我沒要求的部分

## 驗收習慣
- 每次改完先跑 streamlit run app.py 確認可啟動、無 exception
- 回報時列出:改了哪些檔案、為什麼、如何驗證
